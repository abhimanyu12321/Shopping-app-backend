# Shopping-app-backend
To use this as a backend 
Add a config.env file in config folder and add the following variablle in that file:->

PORT,
MONGO_URI,
JWT_SECRET, 
JWT_EXPIRE, 
COOKIE_EXPIRE, 
SMPT_SERVICE, 
SMPT_MAIL,
SMPT_PASSWORD,  
SMPT_HOST,
SMPT_PORT, 
CLOUDINARY_NAME,  
CLOUDINARY_API_KEY , 
CLOUDINARY_API_SECRET
